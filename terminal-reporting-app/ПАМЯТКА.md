# ⚡ Памятка по запуску

## 🎯 Самое главное

### Запустить приложение:
```bash
start.bat   # Windows
```
или
```bash
./start.sh  # Linux/Mac
```

### Остановить приложение:
Нажмите `Ctrl+C` в каждом окне терминала

## 🔧 Полезные команды

### Backend (в папке backend):
```bash
npm run dev              # Запустить backend
npx prisma studio        # Открыть Prisma Studio (GUI для БД)
npx prisma migrate dev   # Создать миграцию
```

### Frontend (в папке frontend):
```bash
npm run dev              # Запустить frontend
npm run build            # Собрать для production
npm run preview          # Предпросмотр production сборки
```

### Docker:
```bash
docker-compose up        # Запустить все (PostgreSQL + Backend + Frontend)
docker-compose down      # Остановить все
docker-compose logs      # Посмотреть логи
```

## 🆘 Решение проблем

### Проблема: Порт занят
**Решение:** Остановите другие процессы или измените порт в конфигурации

### Проблема: База данных не создается
**Решение:**
```bash
cd backend
Remove-Item dev.db -Force
npx prisma migrate dev --name init
```

### Проблема: Зависимости устарели
**Решение:**
```bash
cd backend && npm install && cd ../frontend && npm install
```
