## 🚀 Быстрый старт (Первая неделя)

### Шаг 1: Обновить схему БД
```bash
cd backend
# Скопировать модели из ПЛАН_РЕАЛИЗАЦИИ.md в prisma/schema.prisma
npx prisma migrate dev --name add_vessels_and_berths
npx prisma generate
```

### Шаг 2: Создать API маршруты
```bash
mkdir src/routes
# Создать файлы: vessels.ts, vesselCalls.ts, berths.ts
# Код готов в ПЛАН_РЕАЛИЗАЦИИ.md
```

### Шаг 3: Создать UI страницы
```bash
cd frontend
# Создать файлы: BerthsPage.tsx, VesselCallsPage.tsx
# Код готов в ПЛАН_РЕАЛИЗАЦИИ.md
npm install date-fns
```

### Шаг 4: Обновить навигацию
```typescript
// App.tsx - добавить новые маршруты
<Route path="/vessel-calls" element={<VesselCallsPage />} />
<Route path="/berths" element={<BerthsPage />} />
```
