## 🚀 Как запустить

### 1. Настройка базы данных

```bash
# Создать PostgreSQL базу
createdb terminal_db

# В backend/.env
DATABASE_URL=postgresql://user:password@localhost:5432/terminal_db
```

### 2. Backend

```bash
cd backend
npm install
npm run prisma:generate
npm run prisma:migrate
npm run dev
```

Запустится на `http://localhost:3001`

### 3. Frontend

```bash
cd frontend
npm install
npm run dev
```

Запустится на `http://localhost:5173`
