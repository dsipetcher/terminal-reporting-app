# 🚀 Краткая инструкция по запуску

## ⚡ Самый быстрый способ (SQLite)

### Windows:
```bash
start.bat
```

### Linux/Mac:
```bash
chmod +x start.sh
./start.sh
```

**Готово!** Приложение откроется автоматически.

---

## 🐳 Docker (с PostgreSQL)

```bash
docker-compose up --build
```

Откройте: http://localhost:5173

---

## 🌐 GitHub Pages (Production)

### 1. Backend на Railway.app

```bash
cd backend

# Войти в Railway
npm install -g @railway/cli
railway login

# Создать проект
railway init

# Добавить PostgreSQL
railway add postgresql

# Развернуть
railway up

# Получить URL
railway domain
```

### 2. Настроить Frontend

1. Скопировать URL backend из Railway
2. Добавить в GitHub Secrets:
   - Имя: `VITE_API_URL`
   - Значение: `https://your-app.railway.app/api`

3. Push в main branch - автоматический деплой!

### 3. Настроить GitHub Pages

- Settings → Pages
- Source: gh-pages branch
- Сохранить

Готово! Ваше приложение на: `https://username.github.io/terminal-reporting-app/`
