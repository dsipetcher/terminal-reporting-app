## 🎯 Что это?

Расширение системы учета железнодорожных вагонов до полноценной портовой системы управления грузами, которая обрабатывает:

- 🚢 **Морские суда** и судозаходы
- 🚂 **Железнодорожные вагоны** и составы
- 🚛 **Автотранспорт** с тайм-слотированием
- 📦 **Контейнеры** с адресным хранением
- 🏭 **Причалы** и склады
- 📄 **Документооборот** и интеграции

---

## 🚀 Быстрый старт

### Запуск текущей версии:

```bash
# Backend
cd backend
npm install
npx prisma migrate dev
npm run dev

# Frontend
cd frontend
npm install
npm run dev
```

### Доступ:
- Frontend: http://localhost:5173
- Backend API: http://localhost:3001
