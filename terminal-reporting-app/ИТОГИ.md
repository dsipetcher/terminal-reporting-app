## 🔄 Переключение на PostgreSQL (опционально)

Если понадобится PostgreSQL:

1. Обновить `backend/prisma/schema.prisma`:
```prisma
datasource db {
  provider = "postgresql"
  url      = env("DATABASE_URL")
}
```

2. Запустить Docker:
```bash
docker-compose up
```

Все остальное работает без изменений!
